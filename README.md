# ds-preprocessing-dim-reduction
Data preprocessing and dimensionality reduction experiment using Kaggle and UCI datasets.

# Data Science Fundamentals: Preprocessing & Dimension Reduction

This project covers end-to-end data cleaning and dimensionality reduction experiments.

## Task 1: Data Preprocessing
- Dataset: https://www.kaggle.com/datasets/thedrzee/top-selling-books-data2023-2025
- Steps: Null handling, Standardization, One-Hot Encoding, and Multicollinearity checks.

## Task 2: Farm Ads Dimensionality Reduction
- Compared 3 algorithms (PCA, SVD, etc.) across various dimensions.
- Used Logistic Regression to measure accuracy.

## Task 3: Signal & Consumption Visualization
- Visualized EEG and Repeat Consumption datasets from UCI.
